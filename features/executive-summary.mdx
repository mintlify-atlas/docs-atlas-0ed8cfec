## Overview

The **Executive Summary** is the main landing page of Visor KPI Comercial, providing decision-makers with a complete snapshot of the business performance in a single view. It consolidates sales data, team performance, customer coverage, and active alerts without requiring manual spreadsheet analysis.

## Key Metrics Displayed

### Primary KPIs

The executive summary presents four main KPI cards at the top of the dashboard:

**Sales for the Period**
- Total net sales amount
- Percentage change vs. previous period
- Progress toward sales objective
- Compliance percentage with visual indicator (green/yellow/red)

**Team Compliance**
- Overall team achievement percentage
- Visual gauge showing progress toward 100% goal
- Color-coded status based on performance thresholds

**Gross Margin**
- Total gross profit for the period
- Percentage variation compared to previous period
- Profitability trend indicator

**Customer Coverage**
- Percentage of active customers in the portfolio
- Number of customers served during the period
- New and lost customer metrics

<Note>
All KPIs automatically compare against the previous equivalent period, showing month-over-month or year-over-year variations depending on your date range selection.
</Note>

## Implementation Details

The Executive Summary page is implemented in `visor_kpi/app.py` as the main entry point of the application.

### Data Loading and Filtering

The page loads data using the centralized data loader and applies sidebar filters:

```python
with st.spinner("Calculando indicadores..."):
    data       = load_data()
    df         = get_filtered_data(fecha_desde, fecha_hasta, 
                                    vendedores_sel, zonas_sel, canales_sel)
    df_ant     = get_filtered_data_periodo_anterior(fecha_desde, fecha_hasta, 
                                                      vendedores_sel, zonas_sel, canales_sel)
    clientes_df  = data["clientes"]
    productos_df = data["productos"]
    objetivos_df = data["objetivos"]
```

### KPI Calculation

All metrics are calculated using specialized functions from the KPI engine:

```python
kv    = kpi_ventas_periodo(df, df_ant, objetivos_df, fecha_desde, fecha_hasta)
km    = kpi_margen(df, df_ant)
kc    = kpi_cobertura_clientes(df, clientes_df, df_ant)
kv_df = kpi_por_vendedor(df, objetivos_df, df_ant, fecha_desde, fecha_hasta)
kp_df = kpi_productos(df, df_ant)
kcli  = kpi_clientes(df, clientes_df, df_ant, fecha_hasta)
evo   = kpi_evolucion_mensual(df, objetivos_df)
alertas = get_alertas_activas(df, df_ant, clientes_df, productos_df, 
                                objetivos_df, fecha_hasta)
```

### Rendering KPI Cards

Each KPI card is rendered with consistent styling and behavior:

```python
render_kpi_card(
    label        = "Ventas del Período",
    value        = fmt_moneda(kv["importe_total"]),
    delta_pct    = kv.get("importe_vs_anterior"),
    objetivo     = fmt_moneda(kv.get("objetivo_total", 0)),
    cumplimiento = kv.get("cumplimiento_pct"),
    icon         = "💰",
)
```

## Top Rankings

Three side-by-side rankings highlight the best performers:

**Top Sellers**
- Ranked by total sales and compliance percentage
- Shows position variation vs. previous period
- Color-coded performance indicators

**Top Customers**
- Sorted by total purchase amount
- Displays purchase frequency
- Flags at-risk customers

**Top Products**
- Ordered by sales volume
- Shows category and growth trend

## Active Alerts Panel

The alerts section displays the most critical issues and opportunities:

```python
if not alertas.empty:
    n_alta  = len(alertas[alertas["severidad"] == "alta"])
    n_media = len(alertas[alertas["severidad"] == "media"])
    n_info  = len(alertas[alertas["severidad"].isin(["baja", "info"])])
    
    # Display summary metrics
    c_a.metric("🔴 Críticas",    n_alta)
    c_m.metric("🟡 Medias",      n_media)
    c_i.metric("🔵 Informativas", n_info)
```

Each alert displays:
- Severity indicator (red/yellow/blue icon)
- Alert name and code
- Affected entity (seller, customer, or product)
- Detailed description
- Suggested action

<Note>
The executive summary shows the first 4 active alerts. To view all alerts with filtering and management options, navigate to the **Alerts Center** page.
</Note>

## Usage Tips

<AccordionGroup>
  <Accordion title="How to interpret the compliance gauge">
    - **Green zone (>90%)**: Team exceeding expectations
    - **Yellow zone (70-90%)**: Acceptable performance, room for improvement
    - **Red zone (less than 70%)**: Requires immediate attention and action
  </Accordion>
  
  <Accordion title="Best practices for executive review">
    1. Start with the compliance percentage to assess overall health
    2. Check alerts for any critical issues requiring immediate action
    3. Review the evolution chart to identify trends
    4. Examine top rankings to recognize high performers
    5. Use filters to drill down into specific zones or time periods
  </Accordion>
  
  <Accordion title="Customizing the time period">
    Use the sidebar date picker to select any custom range. The system automatically calculates the equivalent previous period for comparison purposes.
  </Accordion>
</AccordionGroup>
