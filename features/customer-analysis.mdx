## Overview

The **Customer Analysis** page (Análisis de Clientes) provides sophisticated portfolio intelligence using advanced analytics like Pareto 80/20 classification, Gini concentration index, scatter analysis of purchase frequency vs. amount, and automatic risk detection. This view answers critical business questions: Which customers drive most revenue? Who's at risk of churning? Where are growth opportunities?

## Key Metrics

### Customer Coverage KPIs

**Active Customers**
- Number of customers with purchases in the selected period
- Coverage percentage of total assigned portfolio
- Visual indicator of portfolio health

**Coverage Percentage**
- Ratio of active to total assigned customers
- Benchmark against 60% threshold
- Color-coded performance indicator

**New Customers**
- Count of customers making their first purchase
- Growth metric

**Lost Customers**
- Customers who were active in previous period but not current
- Churn metric requiring attention

<Info>
The system defines a "lost" customer as one who purchased in the previous equivalent period but has zero transactions in the current period.
</Info>

The page provides multi-dimensional filtering:

```python
with st.expander("🔍 Filtros adicionales de clientes", expanded=False):
    col_cat, col_can, col_est = st.columns(3)

        estado_filter = st.selectbox("Estado", 
                                     ["Todos", "Activos", "En riesgo", "Sin compra reciente"], 
                                     key="_est_filter")

# Apply additional filters
df_cli_filtrado = kcli.copy()
if cat_filter:
    df_cli_filtrado = df_cli_filtrado[df_cli_filtrado["categoria_pareto"].isin(cat_filter)]
if canal_filter:
    df_cli_filtrado = df_cli_filtrado[df_cli_filtrado["canal"].isin(canal_filter)]
if estado_filter == "En riesgo":
    df_cli_filtrado = df_cli_filtrado[df_cli_filtrado["flag_en_riesgo"] == True]
elif estado_filter == "Sin compra reciente":
    df_cli_filtrado = df_cli_filtrado[df_cli_filtrado["dias_desde_ultima_compra"] > 30]
```

**Insights:**

## Pareto Concentration Metrics

```python
if not kcli.empty:
    conc = get_concentracion_pareto(kcli, "importe_total")
    col_a, col_b, col_c = st.columns(3)
    with col_a:
        st.metric(
            "Clientes categoría A",
            f"{conc['n_entidades_A']} clientes",
            delta=f"{conc['pct_entidades_A']*100:.1f}% del total",
        )
    with col_b:
        st.metric(
            "% ventas categoría A",
            f"{conc['pct_ventas_A']*100:.1f}%",
            delta="objetivo ~80%",
        )
    with col_c:
        st.metric(
            "Índice Gini",
            f"{conc['indice_gini']:.3f}",
            help="0 = distribución perfecta, 1 = concentración total",
        )
```

### Gini Index Interpretation

The Gini coefficient measures portfolio concentration:

- **0.0 - 0.4**: Balanced portfolio, low dependency risk
- **0.4 - 0.6**: Moderate concentration, healthy focus
- **0.6 - 0.8**: High concentration, monitor key accounts closely
- **0.8 - 1.0**: Extreme concentration, business continuity risk

<Warning>
A Gini index above 0.6 triggers an alert (ALT004) in the Alerts Center, recommending portfolio diversification strategies.
</Warning>

## Risk and Opportunity Detection

### At-Risk Customers

Automatically identifies customers likely to churn:

```python
en_riesgo = kcli[kcli["flag_en_riesgo"] == True].copy()
if en_riesgo.empty:
    st.success("✅ Sin clientes en riesgo detectados.")
else:
    render_ranking_clientes(en_riesgo, top_n=10)
```

**Risk criteria:**
- No purchase in 30+ days
- Declining frequency trend
- Significant drop in order value
- Category A or B customers showing negative trends

**Suggested actions:**
- Schedule immediate sales visit
- Contact to identify issues
- Offer promotions or incentives
- Review competitive threats

### Growth Opportunities

Identifies customers with expansion potential:

```python
if "vs_periodo_anterior" in kcli.columns:
    opor = kcli[
        (kcli["categoria_pareto"] == "B") &
        (kcli["vs_periodo_anterior"].fillna(0) > 0.20)
    ].sort_values("vs_periodo_anterior", ascending=False)
    if opor.empty:
        st.info("Sin oportunidades detectadas en este período.")
    else:
        render_ranking_clientes(opor, top_n=10)
```

**Opportunity criteria:**
- Category B customers (mid-tier)
- Growth >20% vs. previous period
- High purchase frequency with moderate amounts

**Suggested actions:**
- Upsell higher-value products
- Increase order frequency
- Promote to category A status
- Assign dedicated account manager

## Customer Detail View

Drill-down into individual customer history:

```python
if not kcli.empty:
    clientes_opciones = kcli["razon_social"].tolist() if "razon_social" in kcli.columns else kcli["id_cliente"].tolist()
    cliente_sel = st.selectbox("Seleccionar cliente para ver detalle:", 
                               clientes_opciones, key="_cli_detalle")

    if cliente_sel:
        cli_row = kcli[kcli["razon_social"] == cliente_sel]
        if not cli_row.empty:
            cli_id_sel = cli_row.iloc[0]["id_cliente"]
            df_cli_det = df[df["id_cliente"] == cli_id_sel].copy()

            col_d1, col_d2, col_d3, col_d4 = st.columns(4)
            with col_d1:
                st.metric("Importe total", 
                         fmt_moneda(cli_row.iloc[0].get("importe_total", 0)))
            with col_d2:
                st.metric("Frecuencia", 
                         f"{int(cli_row.iloc[0].get('frecuencia_compra', 0))} compras")
            with col_d3:
                st.metric("Categoría Pareto", 
                         cli_row.iloc[0].get("categoria_pareto", "—"))
            with col_d4:
                dias = cli_row.iloc[0].get("dias_desde_ultima_compra", 0)
                st.metric("Días sin compra", f"{int(dias)} días")

            # Customer evolution chart
            if not df_cli_det.empty:
                evo_cli = kpi_evolucion_mensual(df_cli_det, n_meses=6)
                if not evo_cli.empty:
                    fig_cli = chart_evolucion_ventas(evo_cli, mostrar_objetivo=False,
                                                      title=f"Evolución de {cliente_sel}")
                    st.plotly_chart(fig_cli, use_container_width=True, 
                                   config={"displayModeBar": False})
```

**Shows:**
- Customer summary metrics
- 6-month purchase trend
- Product mix analysis
- Channel usage
- Contact history

## Integration with Alert System

Customer analysis triggers several automatic alerts:

- **ALT002**: Customer without purchase in last 30 days
- **ALT004**: Portfolio with excessive concentration (Gini >0.6)
- **ALT006**: Customer coverage below 60%
- **ALT007**: High-frequency customer with low ticket (opportunity)
