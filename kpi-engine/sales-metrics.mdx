## Overview

The `kpi_ventas_periodo()` function computes comprehensive sales metrics for a given period, including total revenue, quota compliance, transaction counts, and average ticket size with period-over-period comparisons.

<Note>
  This function is the foundation for the **Executive Summary** dashboard view, providing the top-line metrics that answer "How is the business performing?"
</Note>

## Why It Matters

Sales metrics are the primary indicator of commercial performance. This function enables:

- **Performance tracking** against monthly/quarterly objectives
- **Trend identification** through period comparisons
- **Early warning signals** when actual sales deviate from targets
- **Customer behavior insights** via transaction count and average ticket analysis

## Function Signature

```python kpi_engine.py
def kpi_ventas_periodo(
    df: pd.DataFrame,
    df_anterior: pd.DataFrame | None = None,
    objetivos_df: pd.DataFrame | None = None,
    fecha_desde: date | None = None,
    fecha_hasta: date | None = None,
) -> dict:
    """
    KPIs agregados de ventas para el período.

    Returns dict con:
        importe_total, importe_vs_anterior, objetivo_total,
        cumplimiento_pct, transacciones_count, ticket_promedio
    """
```

## Parameters

<ParamField path="df" type="pd.DataFrame" required>
  Sales DataFrame filtered for the target period. Must contain columns:
  - `importe_neto`: Net sales amount
  - `id_cliente`: Customer identifier
  - `id_venta`: Transaction identifier
</ParamField>

<ParamField path="df_anterior" type="pd.DataFrame | None" default="None">
  Sales DataFrame from the previous period for comparison. When provided, calculates `importe_vs_anterior` (% change).
</ParamField>

<ParamField path="objetivos_df" type="pd.DataFrame | None" default="None">
  Objectives DataFrame with columns:
  - `periodo`: Period date (timestamp)
  - `objetivo`: Sales target/quota amount
  
  Required for quota compliance calculation.
</ParamField>

<ParamField path="fecha_desde" type="date | None" default="None">
  Start date of the analysis period. Used to filter objectives DataFrame.
</ParamField>

<ParamField path="fecha_hasta" type="date | None" default="None">
  End date of the analysis period. Used to filter objectives DataFrame.
</ParamField>

## Return Value

Returns a dictionary with the following keys:

| Key | Type | Description |
|-----|------|-------------|
| `importe_total` | `float` | Total net sales amount for the period |
| `importe_vs_anterior` | `float \| None` | % change vs. previous period (e.g., 0.15 = +15%) |
| `objetivo_total` | `float` | Sum of objectives/quotas for the period |
| `cumplimiento_pct` | `float \| None` | Quota compliance ratio (e.g., 1.05 = 105% compliance) |
| `transacciones_count` | `int` | Number of sales transactions |
| `ticket_promedio` | `float` | Average ticket (sales per unique customer) |
| `clientes_unicos` | `int` | Number of unique customers who purchased |

## Business Logic

The function implements the following calculation logic:

### 1. Total Sales
```python
importe_total = df["importe_neto"].sum()
```

### 2. Transaction Metrics
```python
n_transacciones = len(df)
clientes_unicos = df["id_cliente"].nunique()
```

### 3. Average Ticket
Calculated as total sales divided by unique customers (not by transactions):
```python
ticket_prom = importe_total / clientes_unicos if clientes_unicos > 0 else 0.0
```

### 4. Quota Compliance
Filters objectives for the period and calculates the compliance ratio:
```python
mask = (
    (objetivos_df["periodo"] >= pd.Timestamp(fecha_desde)) &
    (objetivos_df["periodo"] <= pd.Timestamp(fecha_hasta))
)
objetivo_total = objetivos_df[mask]["objetivo"].sum()
cumplimiento = importe_total / objetivo_total if objetivo_total > 0 else None
```

### 5. Period Comparison
```python
if df_anterior is not None:
    imp_ant = df_anterior["importe_neto"].sum()
    if imp_ant > 0:
        importe_vs_ant = (importe_total - imp_ant) / imp_ant
```
