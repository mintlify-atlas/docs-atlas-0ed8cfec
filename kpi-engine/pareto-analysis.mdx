## Overview

The Pareto analysis module (`pareto.py`) implements the 80/20 principle for customer and product segmentation. It automatically classifies entities into A/B/C categories and calculates concentration metrics like the Gini index.

<Note>
  The **Pareto Principle** states that ~80% of results come from ~20% of causes. In commercial terms: **80% of revenue typically comes from 20% of customers**.
</Note>

## Why It Matters

Pareto analysis enables strategic resource allocation:

- **Identify VIP customers/products** that drive the majority of revenue
- **Prioritize sales efforts** on high-impact accounts (Category A)
- **Detect concentration risk** when too few customers control too much revenue
- **Optimize inventory** by focusing on high-rotation SKUs
- **Measure inequality** with the Gini coefficient

<Warning>
  High concentration (Gini > 0.6) indicates **revenue risk**: losing a single A-category customer can significantly impact results.
</Warning>

## Core Functions

The module provides two main functions:

### 1. `calcular_pareto()` - Classification

Adds Pareto columns to a DataFrame, classifying each row as A/B/C.

### 2. `get_concentracion_pareto()` - Metrics

Calculates aggregate concentration statistics including the Gini index.

## `calcular_pareto()`

### Function Signature

```python pareto.py
def calcular_pareto(
    df: pd.DataFrame,
    columna_valor: str,
    columna_id: str,
) -> pd.DataFrame:
    """
    Agrega columnas de Pareto al DataFrame.

    Returns:
        DataFrame con columnas adicionales:
        - participacion_pct   : % del total (0-1)
        - acumulado_pct       : % acumulado ordenado desc (0-1)
        - categoria_pareto    : 'A' (0-80%), 'B' (80-95%), 'C' (95-100%)
    """
```

### Parameters

<ParamField path="df" type="pd.DataFrame" required>
  DataFrame to classify. Can contain customers, products, or any entity to analyze.
</ParamField>

<ParamField path="columna_valor" type="str" required>
  Name of the column containing the value to analyze (e.g., `"importe_total"`, `"cantidad"`)
</ParamField>

<ParamField path="columna_id" type="str" required>
  Name of the column containing the entity identifier (e.g., `"id_cliente"`, `"id_producto"`)
  
  Not directly used in calculation, but useful for documentation purposes.
</ParamField>

### Return Value

Returns the **original DataFrame** with three additional columns:

| Column | Type | Description |
|--------|------|-------------|
| `participacion_pct` | `float` | Individual contribution (e.g., 0.15 = 15% of total) |
| `acumulado_pct` | `float` | Cumulative percentage when sorted descending (0-1) |
| `categoria_pareto` | `str` | Classification: `"A"`, `"B"`, or `"C"` |

**Category definitions:**
- **A**: Cumulative ≤ 80% (the vital few)
- **B**: Cumulative 80-95% (the middle)
- **C**: Cumulative 95-100% (the trivial many)

### Business Logic

```python
# 1. Sort descending by value
df = df.sort_values(columna_valor, ascending=False).reset_index(drop=True)

# 2. Calculate individual participation
total = df[columna_valor].sum()
df["participacion_pct"] = df[columna_valor] / total

# 3. Calculate cumulative percentage
df["acumulado_pct"] = df["participacion_pct"].cumsum()

# 4. Assign category based on cumulative
def _cat(acum: float) -> str:
    if acum <= 0.80:
        return "A"
    elif acum <= 0.95:
        return "B"
    return "C"

df["categoria_pareto"] = df["acumulado_pct"].apply(_cat)
```

## `get_concentracion_pareto()`

### Function Signature

```python pareto.py
def get_concentracion_pareto(
    df: pd.DataFrame,
    columna_valor: str = "importe_total",
) -> dict:
    """
    Métricas de concentración de Pareto.

    Returns:
        dict con:
        - pct_entidades_A  : % de entidades en categoría A
        - pct_ventas_A     : % de ventas que representan
        - n_entidades_A    : cantidad
        - indice_gini      : índice de concentración (0-1)
    """
```

### Parameters

<ParamField path="df" type="pd.DataFrame" required>
  DataFrame that has already been processed by `calcular_pareto()`. Must contain:
  - `categoria_pareto` column
  - The value column specified in `columna_valor`
</ParamField>

<ParamField path="columna_valor" type="str" default="importe_total">
  Name of the column containing the metric to analyze.
</ParamField>

### Return Value

| Key | Type | Description |
|-----|------|-------------|
| `pct_entidades_A` | `float` | % of entities in Category A (e.g., 0.20 = 20%) |
| `pct_ventas_A` | `float` | % of total value from Category A (typically ~0.80) |
| `n_entidades_A` | `int` | Absolute count of A-category entities |
| `indice_gini` | `float` | Gini coefficient (0 = perfect equality, 1 = total inequality) |

<Info>
  **Gini Index Interpretation:**
  - **0.0-0.3**: Low concentration (distributed)
  - **0.3-0.5**: Moderate concentration (typical)
  - **0.5-0.7**: High concentration (risky)
  - **0.7-1.0**: Extreme concentration (dangerous)
</Info>

```python Concentration Alert System
from src.pareto import get_concentracion_pareto

GINI_THRESHOLD_HIGH = 0.6
GINI_THRESHOLD_CRITICAL = 0.7

def check_concentration_risk(df_classified, entity_type="customers"):
    conc = get_concentracion_pareto(df_classified)
    
    if conc['indice_gini'] >= GINI_THRESHOLD_CRITICAL:
        return {
            'severity': 'critical',
            'message': f"🚨 CRITICAL concentration in {entity_type}",
            'gini': conc['indice_gini'],
            'detail': f"{conc['pct_ventas_A']:.0%} from {conc['pct_entidades_A']:.0%}"
        }
    elif conc['indice_gini'] >= GINI_THRESHOLD_HIGH:
        return {
            'severity': 'high',
            'message': f"⚠️  HIGH concentration in {entity_type}",
            'gini': conc['indice_gini'],
            'detail': f"{conc['pct_ventas_A']:.0%} from {conc['pct_entidades_A']:.0%}"
        }
    else:
        return {
            'severity': 'ok',
            'message': f"✅ Healthy distribution of {entity_type}",
            'gini': conc['indice_gini']
        }

# Usage
alert = check_concentration_risk(df_customers, "customers")
print(alert['message'])
print(f"Gini: {alert['gini']:.3f}")
if 'detail' in alert:
    print(alert['detail'])
```

### Gini Coefficient Calculation

The Gini index is calculated using the following formula:

```python
def _calcular_gini(valores: np.ndarray) -> float:
    """Calcula el índice de Gini (0 = igualdad perfecta, 1 = concentración total)."""
    if len(valores) == 0:
        return 0.0
    valores = np.sort(valores.astype(float))
    n = len(valores)
    idx = np.arange(1, n + 1)
    total = valores.sum()
    if total == 0:
        return 0.0
    return float((2 * (idx * valores).sum() / (n * total)) - (n + 1) / n)
```
