## Overview

The mock data generator creates a realistic dataset for testing and demonstration purposes. It generates complete business data for a fictional company (Distribuidora Del Sur S.A.) with 27+ months of transactional history.

<Info>
Location: `data/mock/generate_mock_data.py:1`

Generation time: ~30 seconds
</Info>

```python
def _factor_crecimiento(mes: date) -> float:
    """Factor de crecimiento acumulado: 8% anual ≈ 0.64% mensual."""
    meses_transcurridos = (mes.year - FECHA_INICIO.year) * 12 + (mes.month - FECHA_INICIO.month)
    return (1.0064) ** meses_transcurridos
```

## Generation Process

### Step-by-Step

```python
def main():
    print("Generando datos mock...")

    print("  • Vendedores...")
    df_vendedores = generar_vendedores()          # ~5ms

    print("  • Clientes...")
    df_clientes = generar_clientes(df_vendedores) # ~50ms

    print("  • Productos...")
    df_productos = generar_productos()            # ~10ms

    print("  • Objetivos...")
    df_objetivos = generar_objetivos(df_vendedores) # ~100ms

    print("  • Transacciones (puede demorar ~30s)...")
    df_ventas = generar_transacciones(            # ~30s
        df_vendedores, df_clientes, df_productos, df_objetivos
    )

    validar_y_resumir(df_ventas, df_vendedores, df_clientes, df_productos)
    exportar_excel(df_ventas, df_vendedores, df_clientes, df_productos, df_objetivos)

    print("✅ Generación completada.")
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="ModuleNotFoundError: No module named 'faker'">
    Install missing dependencies:
    ```bash
    pip install -r requirements.txt
    ```
    
    Or install individually:
    ```bash
    pip install faker pandas numpy python-dateutil openpyxl xlsxwriter
    ```
  </Accordion>

  <Accordion title="Permission denied error">
    Ensure the `data/raw/` directory is writable:
    ```bash
    mkdir -p data/raw
    chmod 755 data/raw
    ```
    
    Or run with sudo (not recommended):
    ```bash
    sudo python data/mock/generate_mock_data.py
    ```
  </Accordion>

  <Accordion title="Generation takes too long (>2 minutes)">
    Normal generation time is ~30 seconds. If it takes longer:
    - Check CPU usage (should be ~100% on one core)
    - Reduce date range or number of customers/products
    - Close other applications to free memory
    - Ensure you're using Python 3.9+ for better performance
  </Accordion>

  <Accordion title="Validation warnings">
    Some warnings are acceptable:
    - **⚠️ Transacciones**: 8,000-15,000 range is wide; actual count depends on randomness
    - **⚠️ Estacionalidad**: ±5% variance from 25% is normal
    - **⚠️ Productos bajo rotación**: ±3 products variance is acceptable
    
    Critical errors (❌) indicate data integrity issues and should be investigated.
  </Accordion>
</AccordionGroup>
